# Консольная утилита для обзора и скачивания с сайта releases.1c.ru

Запускается командой: `clidownloader.bat`

Принимает единственный необязательный параметр - любую ссылку начинающуюся на https://releases.1c.ru (на любую вложенную страницу)

Логин и пароль для авторизации устанавливаются в переменных окружения `$LOGIN1C` и `$PASS1C`. Например:
``` PS
$env:LOGIN1C="VasiliyAlibabaevich"
$env:PASS1C="123456789"
``` 
Если переменные не заданы, приложение предложит ввод данных авторизации в консоли.

\> [Пример вывода консоли](doc/console_output.md)
